# Org-Uplift: 宁德时代制造部团队AI能力提升

## Demo 概述

用Org-Uplift框架对宁德时代制造部（~7万人）进行AI能力评估和分层提升规划——从产线操作员到工艺工程师到质量总监，每个层级定制AI赋能路径，目标是从"少数人会用AI"升级为"人人都用AI"。

## 组织画像

### 制造部层级结构

| 层级 | 人数(估) | 典型岗位 | 当前AI使用率 |
|------|----------|----------|-------------|
| L1 产线操作员 | ~50,000 | 涂布操作员、卷绕操作员、注液操作员 | <5% |
| L2 班组长/技师 | ~10,000 | 产线班组长、设备技师、质检员 | ~10% |
| L3 工程师 | ~7,000 | 工艺工程师、质量工程师、设备工程师 | ~25% |
| L4 主管/经理 | ~2,000 | 车间主任、质量经理、工艺经理 | ~30% |
| L5 总监/VP | ~200 | 制造总监、质量总监、工艺总监 | ~20% |

### 各层级AI需求差异

| 层级 | AI需求场景 | 工具复杂度 |
|------|-----------|-----------|
| L1 操作员 | SPC数据自动填报、设备异常语音提醒 | 极简(语音/按钮) |
| L2 班组长 | 班次报表自动生成、产线状态看板 | 低(看板/报表) |
| L3 工程师 | 工艺参数优化、缺陷分析、8D报告 | 中(分析工具) |
| L4 经理 | 产能预测、OEE分析、跨线资源调配 | 中高(决策支持) |
| L5 总监 | 战略决策、投资评估、竞品分析 | 高(多维分析) |

## 评估维度

### AI成熟度评估模型(5级)

| 等级 | 描述 | 制造部占比(估) |
|------|------|---------------|
| Level 0 | 未使用AI | 65% |
| Level 1 | 偶尔使用通用AI(ChatGPT等) | 20% |
| Level 2 | 使用专业AI工具完成具体任务 | 10% |
| Level 3 | 将AI嵌入日常工作流 | 4% |
| Level 4 | 能设计AI解决方案并推广 | 1% |

### 评估问卷设计
- 工具使用频率(每周AI使用时长)
- 场景覆盖度(AI用于哪些工作任务)
- 自主能力(能否独立配置AI工具)
- 推广影响(是否帮助他人使用AI)

## 提升路径设计

### L1操作员 → Level 1目标

```
阶段1(Week 1-2): AI认知培训
  - 什么是AI？宁德时代灯塔工厂AI案例展示
  - 实操：用语音助手查询工艺参数

阶段2(Week 3-4): 工具上手
  - SPC数据智能填报(从手动→语音录入)
  - 设备异常AI提醒的响应流程

阶段3(Month 2-3): 融入日常
  - AI辅助交接班报告
  - 常见缺陷AI识别结果的解读
```

### L3工程师 → Level 3目标

```
阶段1(Week 1-2): 专业AI工具培训
  - Karpathy-KB工艺知识库使用
  - SAM3/Cognitive-YOLO质检系统操作

阶段2(Week 3-4): 分析能力提升
  - 用AI分析涂布参数趋势
  - 用AI生成8D报告初稿

阶段3(Month 2-3): 工作流嵌入
  - 日常工艺优化流程中嵌入AI辅助
  - 建立个人AI提示词库
```

## 演示流程

```
Step 1: 展示制造部组织画像——5级结构、7万人分布
Step 2: 展示AI成熟度评估问卷(现场快速填写)
Step 3: AI自动生成团队AI能力雷达图
Step 4: 根据评估结果生成分层提升方案
Step 5: 展示培训课程安排(在线+线下混合)
Step 6: 展示3个月后的预期AI使用率提升曲线
```
