# PPT-Gen: 宁德时代品牌色PPT自动生成

## Demo 概述

用AI根据宁德时代品牌视觉规范（主色绿#00A651），自动生成专业级演示文稿。支持技术汇报、客户提案、内部培训等多种场景模板，确保13万人的对外演示材料保持品牌一致性。

## 品牌视觉规范

| 元素 | 规范 |
|------|------|
| 主色 | CATL Green #00A651 |
| 辅助色 | 深蓝#003B71、浅灰#F2F2F2、白#FFFFFF |
| 标题字体 | 思源黑体 Bold / Source Han Sans Bold |
| 正文字体 | 思源黑体 Regular |
| Logo位置 | 右上角或左下角，最小尺寸规范 |
| 图表配色 | 绿色系渐变 #00A651→#00C853→#69F0AE |

## 场景模板

### 模板1: 技术汇报
- 灯塔工厂月度AI质检报告
- 封面：宁德时代绿+白底，标题"极限制造AI质检月报"
- 内容页：数据图表为主，良率趋势/缺陷分类/OEE对比

### 模板2: 客户提案
- 向车企客户展示麒麟电池技术方案
- 封面：产品渲染图+绿色渐变背景
- 内容页：CTP3.0技术优势、神行超充参数、产能保障

### 模板3: 内部培训
- BMS研发新人入职培训
- 封面：简洁绿白配色
- 内容页：图文结合，流程图为主

## 演示流程

```
Step 1: 选择场景模板——"技术汇报：灯塔工厂AI质检月报"
Step 2: 输入内容要点——良率99.85%、缺陷检出率99.2%、OEE 87%
Step 3: AI自动生成10页PPT
        ├── 封面(品牌色+标题)
        ├── 目录页
        ├── 数据总览(KPI仪表盘)
        ├── 良率趋势图(折线图，绿色系)
        ├── 缺陷分类饼图
        ├── OEE对比柱状图
        ├── 改进措施(文字+图标)
        ├── 下月目标
        ├── 附录(数据明细)
        └── 封底(Logo+联系方式)
Step 4: 一键调整——切换到客户提案风格
Step 5: 导出PPTX文件
```

## 智能内容填充

- 自动将数字转换为数据可视化（表格→图表）
