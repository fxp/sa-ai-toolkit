# Karpathy-KB: 电池制造工艺知识库

## Demo 概述

基于Karpathy风格的向量化知识库，将宁德时代灯塔工厂的五大核心工艺（极片制造、涂布、卷绕、注液、化成）的know-how结构化存储，让13万员工通过自然语言即时获取精准工艺知识。

## 工艺知识体系

### 五大工序知识范围

| 工序 | 知识领域 | 关键参数 | 常见异常 |
|------|----------|----------|----------|
| 极片制造 | 浆料搅拌、匀浆工艺 | 固含量、粘度、粒径分布 | 团聚、沉降、气泡 |
| 涂布 | 挤压涂布、间歇涂布 | 面密度、涂布速度、烘箱温度 | 厚度不均、边缘翘起、针孔 |
| 卷绕 | 正负极对齐、隔膜张力 | 张力控制、对齐精度、卷绕速度 | 褶皱、错位、极耳变形 |
| 注液 | 电解液注入、真空浸润 | 注液量、真空度、浸润时间 | 注液不足、气泡残留、漏液 |
| 化成 | SEI膜形成、容量分选 | 充放电曲线、EIS阻抗、容量 | SEI膜不均、容量衰减、内阻偏高 |

### 知识来源

- 745位博士的研究成果和工艺论文
- 灯塔工厂SOP文件（数千份）
- MES历史数据中的最佳实践参数
- 8D报告和异常处理案例库
- 第四范式AI平台的工艺优化结论

## 演示流程

```
Step 1: 展示知识库结构——五大工序×参数×异常×处理方案

Step 5: 演示新人培训——"新入职工艺工程师的注液操作要点？"
```

## 知识库技术架构

```
原始文档 → 文本提取+清洗 → 向量化(Embedding)
                                    ↓
用户查询 → 意图识别 → 向量检索 → Rerank → LLM生成答案
                                    ↑
                        结构化知识(参数表/SOP/8D报告)
```
