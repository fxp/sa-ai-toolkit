# Skill-Governance: 13万人AI Skill治理

## Demo 概述

为宁德时代13万人设计AI Skill治理平台——跨研发(22,901人)、制造(~70,000人)、质量和供应链(~37,000人)四大条线，统一管理AI技能的注册、分发、权限、审计，确保"对的人用对的AI做对的事"。

## 治理挑战

13万人规模的AI治理复杂性：
- **四大条线需求差异大** — 研发要文献检索，制造要SPC分析，质量要缺陷检测，供应链要需求预测
- **权限控制** — 工艺参数属核心机密，不同层级/部门的数据访问权限不同
- **合规要求** — ISO 27001信息安全、IATF 16949汽车质量、欧盟AI法案
- **成本控制** — 13万人×LLM API调用，成本需要精细化管理
- **Shadow AI风险** — 员工私自使用外部AI工具，存在数据泄露风险

| L1 通用 | 仅公开数据 | 无需审批 | 月度 |

## 演示流程

```

Step 2: 演示Skill注册流程——新增一个"固态电池文献检索"Skill

Step 6: 展示月度治理报告——成本/合规/使用率/ROI
```
