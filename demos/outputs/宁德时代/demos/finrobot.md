# FinRobot: 宁德时代 vs 比亚迪投资价值分析

## Demo 概述

用FinRobot多Agent金融分析框架，从财务指标、技术壁垒、市场空间、估值模型四个维度，对比分析宁德时代(300750.SZ)和比亚迪(002594.SZ/1211.HK)的投资价值。

## 分析框架

### Agent 1: 财务分析师
| 指标 | 宁德时代(2025) | 比亚迪(2025) |
|------|---------------|-------------|
| 营收 | 4,237亿元(+17%) | ~7,600亿元(+20%+) |
| 净利润 | 722亿元(+42.28%) | ~400亿元 |
| 净利率 | ~17% | ~5.2% |
| 研发投入 | 221亿元 | ~500亿元 |
| 动力电池市占率 | 39.2% | ~17% |

### Agent 2: 技术分析师
- 宁德时代技术栈：CTP3.0、麒麟电池、神行超充、骁遥增混、天恒储能
- 比亚迪技术栈：刀片电池、CTB(Cell to Body)、e平台3.0
- 共同前沿：固态电池、钠离子电池

### Agent 3: 市场分析师
- 全球动力电池市场规模：2025年~1.2万亿元，2030年预计~3万亿元
- 储能电池市场：2025年~3000亿元，年增速40%+
- 电池回收市场：2030年预计~1500亿元

### Agent 4: 估值建模师
- DCF模型：基于661GWh产能和产能利用率假设
- 相对估值：P/E、EV/EBITDA与LG新能源、三星SDI对比
- 情景分析：碳酸锂价格、固态电池时间表、政策变化

## 演示流程

```
Step 1: 输入两家公司最新财报数据
Step 2: Agent自动拉取行业数据和竞品信息
Step 3: 多维度对比分析生成（财务/技术/市场/估值）
Step 4: 投资建议：Bull/Base/Bear三种情景
Step 5: 风险因素排序和敏感度分析
```

## 核心对比维度

1. **商业模式差异** — 宁德时代平台化(纯电池) vs 比亚迪垂直整合(整车+电池)
2. **利润质量** — 宁德时代17%净利率 vs 比亚迪5%净利率（含整车业务）
3. **增长引擎** — 储能(天恒)+回收(邦普) vs 整车出口+储能
4. **估值逻辑** — 制造业龙头溢价 vs 新能源车全产业链估值

## 预期产出

- 双方财务指标雷达图对比
- 2025-2030年盈利预测模型
- Bull/Base/Bear三种情景下的目标价
- 关键风险因子排序

## 演示话术要点

- 直接引用宁德时代722亿净利润、42.28%增速等最新数据
- 突出661GWh装机量和39.2%市占率的规模优势
- 分析麒麟电池/神行超充的技术溢价能力
