# AI-Tools: 为宁德时代推荐AI工具组合

## Demo 概述

根据宁德时代的业务场景、技术栈和组织规模，为四大条线（研发/制造/质量/供应链）推荐最适合的AI工具组合，并设计集成架构。

## 推荐原则

1. **私有化部署优先** — 电池制造工艺属核心机密，大模型不出厂区
2. **与现有技术栈兼容** — 已有第四范式AI平台，需协同而非替换
3. **规模化覆盖** — 13万人使用，需要考虑成本和易用性
4. **合规性** — 满足欧盟电池法规、ISO 26262(ASIL-D)等合规要求
