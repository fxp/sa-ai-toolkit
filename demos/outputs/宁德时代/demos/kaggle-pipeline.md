# Kaggle-Pipeline: 电池寿命预测竞赛级Pipeline

## Demo 概述

用Kaggle竞赛级方法论为宁德时代构建电池寿命预测模型——从化成阶段数据预测电池循环寿命，将预测误差从±8%压缩到±3%，支撑天恒储能"5年零衰减"承诺的数据验证。

## 业务价值

- **天恒储能系统** — "5年零衰减"是宁德时代储能的核心卖点，需要精准寿命预测支撑质保承诺
- **质量分选** — 化成阶段预测寿命，提前筛选不合格电芯，避免流入PACK环节
- **客户定价** — 寿命预测精度直接影响储能系统的质保成本和定价策略

## 数据来源

```
化成数据:
  - 首圈充放电曲线 (V/I/T vs Time, 采样频率1Hz)
  - EIS阻抗谱 (0.01Hz-100kHz)
  - 容量/内阻首圈数据
  - 化成温度曲线
  - OCV(开路电压)弛豫曲线

标签数据:
  - 循环寿命 (到80%SOH的充放电次数)
  - 衰减曲线 (容量 vs 循环次数)
```

## Pipeline架构

```
原始数据 → 特征工程 → 模型训练 → 集成学习 → 验证
    │           │           │          │         │
    │     ┌─────┤     ┌─────┤    ┌─────┤   ┌─────┤
    │     │充放电  │     │XGBoost│    │Stacking│  │K-Fold
    │     │曲线特征│     │LightGBM│   │Blending│  │时间序列
    │     │EIS特征 │     │LSTM    │   │加权平均│  │交叉验证
    │     │统计特征│     │TabNet  │   │        │  │
    │     └──────┘     └──────┘   └──────┘  └──────┘
```

## 特征工程

### 充放电曲线特征
| 特征类别 | 具体特征 | 数量 |
|----------|----------|------|
| 容量特征 | 首圈容量、充放电效率、dQ/dV峰位 | 12 |
| 电压特征 | CC/CV转折点、电压平台长度、IR drop | 8 |
| 温度特征 | 最高温度、温升速率、温度不均匀度 | 6 |
| 时间特征 | 化成总时间、各阶段时间、弛豫时间 | 5 |

### EIS阻抗谱特征
| 特征类别 | 具体特征 | 数量 |
|----------|----------|------|
| 等效电路 | Rs(欧姆内阻)、Rct(电荷转移电阻)、Warburg系数 | 6 |
| 频率特征 | 特征频率、低频斜率、高频截距 | 4 |

## 演示流程

```
Step 1: 展示化成数据样本——一组麒麟电池电芯的化成曲线
Step 2: 自动特征提取——从原始曲线提取36维特征
Step 3: 模型训练——XGBoost/LightGBM/LSTM三模型并行
Step 4: 集成学习——Stacking融合，展示预测vs实际寿命散点图
Step 5: 特征重要性分析——哪些化成参数最影响寿命
Step 6: 在线推理演示——新电芯化成完成后实时预测寿命
```
