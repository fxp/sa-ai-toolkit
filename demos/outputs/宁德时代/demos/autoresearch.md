# AutoResearch: 固态电池电解质材料自动科研

## Demo 概述

用AI自动化科研工作流，为宁德时代研发中心加速固态电池关键材料——硫化物电解质的文献检索、筛选、对比分析和综述生成。将一篇材料筛选报告从2周缩短到24小时。

## 研发背景

宁德时代221亿元研发投入的重点方向之一：
- **固态电池** — 下一代电池技术，能量密度提升50%+，安全性大幅提升
- **硫化物电解质** — 离子电导率最高(>10mS/cm)，但空气稳定性和界面问题待解
- **竞争态势** — 三星SDI宣布2027年全固态量产，丰田、宝马押注重投
- **宁德时代策略** — 凝聚态电池(半固态过渡) + 全固态电池并行研发

## 科研任务设定

```yaml
研究课题: 硫化物固态电解质材料筛选
目标指标:
  - 离子电导率: >10 mS/cm (室温)
  - 电化学窗口: >5V
  - 空气稳定性: >24h不分解
  - 界面阻抗: <50 Ω·cm²
文献范围: 2022-2025年, Nature/Science/AEM/JACS/JES等
输出要求: 材料对比表 + 技术路线建议 + 专利态势分析
```

## AI科研工作流

```
Step 1: 文献检索
        ├── 学术数据库(Google Scholar/Scopus/WoS)
        ├── 专利数据库(USPTO/EPO/CNIPA)
        └── 预印本(arXiv/ChemRxiv)

Step 2: 文献筛选
        ├── 关键词过滤: sulfide solid electrolyte + ionic conductivity
        ├── 指标过滤: 离子电导率>10mS/cm
        └── 质量过滤: IF>10期刊优先

Step 3: 信息提取
        ├── 材料成分和合成方法
        ├── 性能指标(电导率/窗口/稳定性)
        ├── 测试条件(温度/压力/对称电池)
        └── 作者和机构信息

Step 4: 对比分析
        ├── 材料性能雷达图
        ├── 技术路线对比(硫化物/氧化物/聚合物)
        └── 成本和可量产性评估

Step 5: 报告生成
        ├── 材料筛选排名表
        ├── 技术路线建议
        ├── 专利布局建议
        └── 下一步实验方案
```

## 演示流程

```

Step 2: AI自动检索相关文献(展示检索过程)

```
