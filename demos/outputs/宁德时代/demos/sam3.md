# SAM3: 电池表面缺陷检测 + SAM-Audio电芯声学检测

## Demo 概述

双模态AI质检方案：SAM3(Segment Anything Model 3)用于电池极片和电芯表面的视觉缺陷检测，SAM-Audio用于电芯内部缺陷的超声波声学检测。两种模态交叉验证，将宁德时代极限制造的漏检率再降一个数量级。

## 极限制造质检挑战

宁德时代的PPB级(十亿分之一)缺陷率目标：
- 661GWh年装机量 ≈ 每天数亿个质检点
- 传统AOI(自动光学检测)漏检率约0.3-0.5%
- 表面缺陷AOI可检，但内部缺陷(气泡/分层/电解液分布不均)无法通过视觉发现
- PPB级目标要求视觉+声学双重保障

## 模态1: SAM3视觉检测

### 检测范围

| 检测工位 | 缺陷类型 | 检测精度 |
|----------|----------|----------|
| 涂布后极片 | 厚度不均、针孔、气泡、边缘翘起 | 分辨率<10um |
| 卷绕后电芯 | 对齐偏移、褶皱、极耳变形 | 对齐精度<0.1mm |
| 封装后电芯 | 外壳划伤、焊缝缺陷、密封不良 | 焊缝检测率>99.5% |
| PACK组装 | CTP模组间距、连接器状态、热管理胶路 | 间距精度<0.5mm |

### SAM3优势
- **零样本分割** — 无需为每种缺陷单独训练，输入prompt即可分割任意缺陷区域
- **实例级分割** — 同一张图中区分多个独立缺陷，精确计数
- **边缘精度** — 缺陷边界精度远超传统阈值分割，支持面积/形状精确测量

## 模态2: SAM-Audio声学检测

### 检测原理
```
超声波激励 → 电芯声学响应 → 频谱分析 → 缺陷识别
                                    ↓
正常电芯: 特征频率稳定，谐波分布均匀
缺陷电芯: 频率偏移/异常谐波/能量衰减
```

### 内部缺陷检测

| 缺陷类型 | 声学特征 | 传统方法 |
|----------|----------|----------|
| 内部气泡 | 高频衰减异常 | X-Ray(慢/贵) |
| 极片分层 | 谐波分布异常 | 破坏性拆解 |
| 电解液分布不均 | 阻尼系数偏差 | 无法检测 |
| 内部短路隐患 | 特征频率偏移 | 充放电筛选(慢) |
| 焊接虚焊 | 能量传递异常 | 拉力测试(抽检) |

## 双模态融合

```
                    ┌─── SAM3视觉 ─── 表面缺陷置信度
电芯质检 ──→ 融合引擎 ─┤
                    └─── SAM-Audio ── 内部缺陷置信度
                              ↓
                    联合判定 → 综合质量分数 → PASS/FAIL/复检
```

## 演示流程

```
Step 1: 展示一张电池极片图片，SAM3自动分割标记缺陷区域
Step 2: 展示缺陷分类和尺寸测量结果
Step 3: 切换到声学检测——播放电芯超声波响应频谱
Step 4: SAM-Audio识别内部异常信号
Step 5: 双模态融合判定——一个表面OK但内部有气泡的电芯案例
Step 6: 展示质检报告自动生成
```

## 部署建议

- SAM3视觉模块部署在涂布后、卷绕后、封装后三个工位
- SAM-Audio声学模块部署在化成前和PACK装配前两个工位
- 融合引擎部署在边缘计算节点，实时联合判定
