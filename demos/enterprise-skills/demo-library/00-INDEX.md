## 演示顺序建议原则

1. **开场Demo**（前2个）：冲击力最大、最易理解的，PPT生成(10)和看板(04)是首选
2. **中段Demo**（3-7个）：行业最契合的核心Demo，按痛点匹配度排序
3. **收尾Demo**（最后1-2个）：强化信任的Demo，监督员(05)和知识库(02)是首选
4. **必选Demo位置**：知识库(02)放中段，监督员(05)放中后段，PPT生成(10)放开场或压轴
