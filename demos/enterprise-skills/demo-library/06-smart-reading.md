## 核心价值主张

信息密度越来越高，但人的阅读时间是固定的。AI帮你预读文档，用4种类型标注出关键信息，你只需要审核标注，而不是通读全文。

## 4类标注类型（通用，不随行业变化）

```
🔴 关键数据/强制条款  — 必须知道的硬性要求
🟡 非共识/值得怀疑   — AI认为有疑问的表述
🟢 可操作策略        — 可以立即应用的建议
💬 AI批评            — AI对文章逻辑或立场的质疑
```
