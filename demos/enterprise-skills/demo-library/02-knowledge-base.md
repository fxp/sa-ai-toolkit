## 核心价值主张

用Karpathy三文件夹法（raw/→wiki/→outputs/），让AI把散乱文档编译成可查询的结构化知识库。老员工的经验不再随离职消失，新员工第一天就能提专业问题。

## 三文件夹结构（通用）

```
{{知识库名称}}/
├── CLAUDE.md      ← Agent行为指南（填充{{规则说明}}）
├── raw/           ← 原始资料（{{文档类型}}，不可更改）
├── wiki/          ← AI自动编译的结构化知识
└── outputs/       ← AI生成的分析报告
```

## 演示流程（通用结构）

1. `tree .` → 展示三文件夹结构（15秒）
2. `cat CLAUDE.md` → 展示AI行为指南（15秒）
3. Claude Code：`> 读取raw/中所有文件，编译到wiki/，每个概念一个文件，互相链接` （60秒）
4. 提问：`> {{问题示例}}？具体到{{精准答案}}的格式` （30秒）
5. 结果：AI精准回答+引用原文章节 （15秒）
6. `> 体检` → 发现知识盲区{{知识盲区}} （15秒）
