# SA AI Toolkit

AI解决方案架构师的工具包。三个命令覆盖SA全流程。

## 命令

| 命令 | 作用 | 输入 | 输出 |
|------|------|------|------|
| `/gen` | 生成方案 | 企业名称 | 全套定制化Demo演示包 |
| `/customize` | 修改方案 | 修改指令 | 更新后的Demo文件 |
| `/present` | 交付方案 | 导出格式 | PPT / Word / 启动Web演示 |

## 典型工作流

```
/gen 招商银行           → 5分钟，生成21个文件
/customize 角色扮演加到5级 → 修改1个文件
/customize 删掉电路设计   → 更新选型和日程
/present 导出PPT         → 生成.pptx
/present 启动演示         → 打开Org-Uplift Game
```

## 安装

在Claude Code / Claude Cowork中安装 `sa-ai-toolkit.plugin` 文件。
