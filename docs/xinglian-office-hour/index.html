它和通用创业导师有什么不同

产业链视角

把你的项目定位到「算力 → 模型 → 工具链 → 应用」产业链的某一层。不是问「市场大不大」，是问「你占的那个位置，产业链上下游依赖你吗？」

默认 B 端优先

2024 中国大模型市场 53 亿，B 端贡献 47 亿（近 90%）。如果你在做纯 C 端，导师开局就直接开火，不留情面。

大厂防御问题 (Q7)

「如果字节 / 阿里 / 腾讯 / 华为明天做一样的事，你还剩什么？」不是「我们更快」，不是「我们更懂用户」，要一个具体答案。
